# Final DS Project: NBA Team Performance
### Goal : Analyze NBA teams performance over the past decade (2015-2016 season -> 2025-2026 season) to answers various questions
    - Are teams carried by star players?
    - Is there a peak age in a player's career?
    - Do trades improve | degrades a teams/players performance?
